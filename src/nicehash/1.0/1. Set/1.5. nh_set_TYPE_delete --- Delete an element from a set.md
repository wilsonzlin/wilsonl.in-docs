# Signatures

```c
int nh_set_TYPE_delete( nh_set_TYPE_t set, ELEM_T elem )
```

# Arguments

|Name|Description|
|---|---|
|set|The set to modify.|
|elem|The element to remove.|

# Returns

- `1` if an element was deleted (i.e. existed before deletion), `0` otherwise
