# Signatures

```c
int nh_map_TYPE_delete( nh_map_TYPE_t map, KEY_T key )
```

# Arguments

|Name|Description|
|---|
|map|The map to modify.|
|key|The entry to remove.|

# Returns

- `1` if an entry was deleted (i.e. existed before deletion), `0` otherwise
